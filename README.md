2 files: 

student.sql : Creats and populates the students table
A3Q1.py : the application that performs CRUD operations on the recently created SQL table

Steps for execution:

1. navigate to pgAdmin4. Create a new database and navigate to the query tool. Upload student.sql to create and populate the students table. 
2. Type in the terminal where the python script is located: python A3Q1.py or python3 A3Q1.py

The program will prompt you to select 1 out of 5 options: Each option later instructs you on what type of information to input.

1) get all students 
2) add a student 
3) update student email 
4) delete student 
5) exit
